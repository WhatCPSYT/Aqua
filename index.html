Frequently Asked Questions

What is Aqua?
Aqua is an advanced, all-in-one Discord bot built to handle your server configuration seamlessly. It features advanced moderation utilities, high-fidelity music utilities with custom filters, interactive ticket management systems, secure verification portals, and powerful AutoMod layers.
Is Aqua free to use?
Yes! All base components (such as standard verification gates, standard tickets, fundamental auto-moderation settings, and crystal-clear music playbacks) are completely free. Advanced automation layers and structural clean utilities can be unlocked via Aqua Plus.
Is Aqua safe and secure?
Absolutely. Aqua treats data privacy with utmost priority. All stored databases are fully secured and encrypted, and we strictly enforce a zero third-party disclosure policy. Chat logs are scanned in live system memory for AutoMod evaluations and instantly deleted within seconds.
Why does the bot occasionally go offline?
Aqua is currently operated as a localized passion project hosted on a dedicated mini PC setup. Due to this home-hosting nature, occasional local power spikes, ISP network distributions, or brief engine script reboots (taking roughly 3 seconds) can create short windows of downtime.
