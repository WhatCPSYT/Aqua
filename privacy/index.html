Privacy Policy

Your privacy is important to us. Last updated: July 2026

Data Collection

Aqua Bot collects minimal data necessary to function and store your server configurations. This includes:

  • Server & User IDs: Collected to store custom server configurations, log systems, and individual Aqua Plus tier status.
  • Moderation Logs: Only the specific Channel ID designated by server administrators to receive automated logs is saved.
  • Ticket System: Configuration details such as customized Ticket Titles, Descriptions, and setup paths.
  • Verification System: Setup data including customized Verification Titles, Descriptions, and designated Role IDs.

Message Content & AutoMod

No chat messages from your server are permanently collected, read, or saved. When the AutoMod feature is enabled, message content is analyzed in volatile memory for a few seconds solely to check for rule violations. Once verified, this content is instantly and completely wiped.

Data Security & Usage

All data collected by Aqua Bot is fully secured, encrypted, and is absolutely never shared, sold, or distributed to third parties. We use this information strictly to provide configuration permanence, moderation utility, music playback, and technical support.

Data Retention & Permanent Deletion

Configurations are stored safely as long as the bot remains on your server. Server administrators have the ability to completely delete all custom server data (including Ticket setups, Mod Logs, and Verification histories) at any time. This action is permanent and completely irreversible; wiped data cannot be recovered or restored afterward.
