Basic Setup Guide

1. Inviting the Bot

Click the invite link on the main page. Ensure you grant the bot the recommended administrative permissions so all custom modules (like AutoMod and moderation actions) work without conflicts.

2. Checking Prefix / Slash Commands

Aqua operates fully on Discord's modern Slash Command architecture. Simply type the following in any text channel to browse available menus:


/help

Ticket System Setup

Creating a Ticket Panel

Aqua lets you create modern, interactive button panels for user inquiries. Run the ticket creation prompt inside your designated support channel:


/ticket setup

Transcripts & Logs

Whenever a user or staff member closes a ticket using the interactive interface, Aqua automatically generates a secure chat log summary and forwards it straight to your DM.

Music Settings

How to stream Music

Join any active voice channel in your community, jump into a text feed, and pull the bot using the standard song loader:


/play [Song Name or Link]

Unlocking Audio Filters

Subscribers of Aqua Plus get instant access to premium distortion matrices. Tweak the audio quality on the fly with choices like:


/filter bassboost   or   /filter 8d

Verification Gate

Securing your Server Against Raids

Prevent rogue bot accounts and spam entries from overtaking your server. You can initialize a visual prompt system that gates entry until a button click is triggered:


/verify setup
